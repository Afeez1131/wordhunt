Game Rules and Regulation

  1. Objective: Form valid English words from a given base word within a limited time.
  2. Game Setup: Join the game room with a unique username. A host manages the game.
  3. Word Formation: Use letters from the base word to create valid English words. Rearrange letters and use each letter only once.
  4. Word Validation: Submit words for validation after the time is up. Invalid or misspelled words won't be counted.
  5. Scoring: Earn points for valid words formed.
